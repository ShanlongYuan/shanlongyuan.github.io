Hugo PaperMod博客搭建教程

2025-03-13 · 3 分钟 · Shanlong Yuan

我来帮您使用 Hugo 搭建一个使用 PaperMod 主题的博客。以下是具体步骤：

  1. 首先安装 Hugo（如果尚未安装）。打开 PowerShell 或命令提示符，使用以下命令：
1

winget install Hugo.Hugo.Extended
  1. 创建新的 Hugo 站点：
1

hugo new site . --force
  1. 初始化 Git 仓库并添加 PaperMod 主题：
1

git init

1

git submodule add --depth=1 https://github.com/adityatelange/hugo-PaperMod.git themes/PaperMod

  1. 如果出现错误需要先清理残留的 Git 配置。以下是针对 Windows PowerShell 的完整修复步骤：

    1. 删除残留的 Git 配置：
    1
2
3

    Remove-Item -Force -Recurse .git
Remove-Item -Force themes/PaperMod -Recurse -ErrorAction SilentlyContinue
Remove-Item -Force .gitmodules -ErrorAction SilentlyContinue
    1. 重新初始化 Git 仓库：
    1

    git init
    1. 添加 PaperMod 主题子模块：
    1

    git submodule add --depth=1 https://github.com/adityatelange/hugo-PaperMod.git themes/PaperMod

  2. 创建基础配置文件。我们需要创建一个 config.yml 文件：

  1. 创建第一篇文章：
1

hugo new posts/my-first-post.md

这将在 content/posts 目录下创建一个新的 Markdown 文件。

  1. 启动 Hugo 开发服务器：
1

hugo server -D

现在您可以在浏览器中访问 http://localhost:1313 查看您的博客了。
